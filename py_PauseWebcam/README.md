# PauseWebcam.py

A tiny python script to pause your webcam at the press of a button! 
## Installation



Use the package manager [pip](https://pip.pypa.io/en/stable/) to install OpenCV (Open Computer Vision).

```bash
pip install opencv-python
```

## Usage

Simply execute the python script to start your webcam. 

Pressing ` (backquote/backtick) will cause the camera to pause.

Pressing X will cause the camera to stop.

```python
$ py PauseWebcam.py
```
**Ensure that your terminal is executing from the same directory as the python script**
